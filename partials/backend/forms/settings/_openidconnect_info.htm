How to retrieve your credentials

This is a generic OpenID Connect handler. You will need to select an ID provider that supports OIDC and get the required credentials.

  1. Using your selected ID Provider, create an application for authentication
  2. Get the application's authentication domain and OIDC conformant API base URI (ID Provider)
  3. Get the application's Client ID (App ID)
